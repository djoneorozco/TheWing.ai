About the Air Force PT Calculator

PCSUnited’s Air Force PT Calculator helps users estimate their Physical Fitness Assessment score using current Air Force standards. The calculator includes body composition, muscular strength, core endurance, and cardio scoring for the 2.0 mile run and 20m HAMR.

Users can adjust age group, gender, height, waist, strength event, core event, and cardio event to estimate a projected score and identify the minimum passing standards for each supported component.

Air Force PT Calculator FAQ

How is the Air Force PT score calculated?

The total score combines body composition, strength, core, and cardio performance based on age- and gender-based standards.

Does this calculator include the 2.0 mile run and 20m HAMR?

Yes. The calculator supports both the 2.0 mile run and 20m HAMR cardio events.

Can I use this Air Force PT Calculator to estimate passing standards?

Yes. The calculator shows best values and minimum passing values for each supported component.
